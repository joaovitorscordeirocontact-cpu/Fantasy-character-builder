# Fantasy Character Builder

A responsive fantasy character creator made with Vanilla JavaScript.

## Features

* Character type selection
* Responsive UI
* Character customization
* Animated final character screen
* Dynamic DOM manipulation

## Technologies

* HTML
* CSS
* Vanilla JavaScript

## Goals

This project was created to practice:

* DOM manipulation
* Responsive design
* State management
* Dynamic rendering
* JavaScript architecture

## Future Features

* Save character
* More customization options
* Animations
* Character stats system
